# 🏏 IPL Analytics Dashboard | Power BI

An interactive Power BI dashboard for analyzing Indian Premier League (IPL) statistics with dynamic DAX measures, player insights, and season-wise analytics.

## Features

- 📊 Season-wise IPL Analysis
- 🟠 Orange Cap Holder
- 🟣 Purple Cap Holder
- 🏏 Most Fours & Sixes
- ⭐ Player Images
- 🏆 Winning Team Analysis
- 📈 Batting & Bowling Statistics
- 🎯 Interactive Slicers
- ⚡ Advanced DAX Measures

## Tools Used

- Power BI
- DAX
- Power Query
- CSV Dataset
